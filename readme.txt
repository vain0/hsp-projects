
	FifPz

対応：Windows XP
公開：2009 2/15 (Sun)
更新：2008 2/15 (Sun)
 DL ：http://prograpark.ninja-web.net/
　　　→ ソフト・モジュールのたまり場

・概要
	画像を使った15パズルです。右下が欠けますが、仕様です。
	
・使い方
	画像を開き、十字キーボタンで画像の断片をスライドさせます。
	以上。正解しても何もおきません。やったね！
	
	画像を開くには、画面にファイルをドロップするか、
	メニューバー→ファイル→開くで選択するか、Ctrl + O で
	選択してください。
	
	メニューの断片数を変更すると、15パズル以外に
	3パズルや48パズルみたいになります。
	
・レジストリ
	全くいじりません。
	
・キーボード・ショートカット
	[Ctrl + ?]
	　O : 開く
	　C : 閉じる
	　S : 断片をランダムに再配置
	　A : 答えの配置
	　
	　Q : Quit 終了する
