
＠概要
	これは拡張ランタイム作成のサンプルです。
	＋参照：http://prograpark.ninja-web.net/HSP/hpi/exRt.html
	
	なお、hsp のコードは、OpenHSP の rev462 を使用しています。
	
＠内容
	+ jump 群
		- jump.hpi
			jump 命令を使用できるようになるプラグイン。
		- jump.as
			jump.hpi を利用するために include するヘッダ。
		- jump_sample.exe
			jump.hpi の、実行形式のサンプル。同じフォルダに jump.hpi が必要。
			
	+ rtjump 群
		- rtjump.exe
			スクリプトエディタ上で実行するとき(デバッグモード)の拡張ランタイム。
			他の機能に加えて、jump 命令が使用できる。
			"hsp3.exe" と同じフォルダにコピーする。
		- rtjump.hrt
			実行形式(exe)を作るときに必要な拡張ランタイム。
			"runtime" フォルダにコピーする。
		- rtjump.as
			拡張ランタイム rtjump の機能を使うためのヘッダ。
		- rtjump_sample.hsp
			拡張ランタイムのサンプルスクリプト。
		- rtjump_sample.exe
			実行形式のサンプル。単体で jump 命令が機能する。
			実行形式のコンパイルには hspcmp.openhsp-462.dll を使用している
			( 通常のコンパイラでは作成できない )。
		
＠命令について
	jump 命令は、gosub と同じ動作をする命令です。
