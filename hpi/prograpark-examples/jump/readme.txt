＠概要
	サブルーチンから戻り値を簡単に受け取るプラグインです。
	通常、return ではシステム変数 stat, refstr, refdval の
	どれに格納されたかを、呼び出し側が知るすべはありません
	でした。( 種類を決めつけると、一つの型の値しか返せない )
	
	jump 命令を使うと、戻り値を変数に返してくれるため、
	そんなことで迷う必要はありません！
	さらに、jump()関数を使うと、サブルーチンを関数形式で
	呼び出すことが出来ます。
	
	サブルーチンを100倍楽しみたいなら、jump を使いましょう！
	
＠対象
	HSP3限定です。それ以前のバージョンでは使用できません。
	HSP3.1正式版で動作を確認しました。
	
＠環境
	Windows 向けです。WindowsXP SP3 での動作のみ確認しました。
	他のＯＳで動作が確認できた方は、フィードバックをしていただけると
	うれしいです。
	
＠使用方法
　・準備
	hsp3.exe があるフォルダに、jump.hpi をコピーします。
	実行ファイルで使用するときは、jump.hpi を実行ファイルと
	同じフォルダの中に入れてください。
	
	また、使用するときは、スクリプトの始めに
		#include "jump.as"
	と書いておいてください。これがなければ、プラグインを使用できません。
	
　・命令の説明
	jump ラベル, 変数
	
	ラベルに gosub ジャンプして、戻り値を変数に代入します。
	変数は省略可能なので、単純に gosub の代替にもなります。
	
　・関数の説明
	jump( ラベル変数 )
	
	ラベルに gosub ジャンプします。戻り値は、サブルーチンが
	return で返した値と同じです。
	HSP3.1では、関数のパラメータにラベルを書くとエラーになる
	ため、一旦変数に代入する必要があります。
	
	(例)
		mes jump(*lb)	; ダメ
		lb_x = *lb	; ラベルを変数に代入
		mes jump(lb_x)	; ＯＫ
	
＠ソースコード
	srcフォルダの中身はプラグインのソースコードです。Ｃ＋＋。
	不要ならフォルダごと削除してもかまいません。
	また、コンパイルには別途 HSPSDK が必要です。
