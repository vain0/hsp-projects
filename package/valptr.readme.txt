＠概要
	値へのポインタを生成する拡張プラグインです。
	
	関数へ任意の型の値を渡すときなどに使えます。
	ポインタの生存期間は長くないので、早めに使ってください。
	
	詳しくは valptr.as 内のサンプルを参照。
	
	@ いわゆる int* や char* (sptr) のようなものとは違います。
	
＠導入
	ダウンロードした圧縮ファイルを、適当なところに解凍してください。
	他に特別な手続きは必要ありません。
	
＠除去
	関係のあるファイルやフォルダをそのまま削除してください。
	レジストリなどは弄りません。
	「＠パッケージ」参照。
	
＠パッケージ
　　[valptr]
　　　　┣ [src]      : HPIのソースコード (C++)
　　　　┣ valptr.as  : 専用ヘッダ
　　　　┣ valptr.hpi : プラグイン
　　　　┗ readme.txt : このファイル。取扱説明書
　　　　
＠ソースコード
	[src]フォルダの内部のファイルすべてです。不要な場合は、フォルダごと削除して
	もかまいません。
	
	VisualC++ 2008 Express Edition (9.0)を使用しています。言語はC++です。バグや
	間違いを見つけましたら、報告していただけると非常にありがたいです。
	( 「＠参照」を参照 )
	
	また、コンパイルするには hspsdk が必要です。「＠参照」の「HSPTV!」から入手し
	てください。
