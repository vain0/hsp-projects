# AutoLauncher
いくつかのファイルを自動で実行するランチャ。

* 作者の以前のパソコンでは、スタートアップから起動された常駐アプリのタスクトレイアイコンが表示されないことがあったので、これを使ってタイムラグをかける必要があった。

## 要修正
* 実行されるファイルの有無を事前に調べているが、`exist` 命令ではシステムフォルダなどを検索パスに含まないので、みつかるはずのファイルを起動できないことがある (``notepad.exe`` など)。
  * また、ファイルではないものの関連づけ実行も失敗する (URLなど)。
