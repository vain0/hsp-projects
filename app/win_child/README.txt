
	WinChild
	
＠概要
	WinChild は、他のアプリケーションのウィンドウ (特に Overlapped Window) を、
	1つのウィンドウの中に無理矢理閉じこめるソフトです。
	@ Windows自動更新後の「早く再起動しろやゴラァ」ウィンドウを黙らせる、なども。
	
	WinNegator は、基本的に WinChild と同じですが、これは他のウィンドウを確実に
	抹殺するためのソフトです。たまにある、タスクマネージャでのプロセス削除もでき
	ないようなプログラムでも、これで捕獲してから削除すれば、通常通り削除できます。
	
	なお、一部のアプリケーションは終了時にクラッシュしますので、ご注意ください。
	ご利用は自己責任でお願いします。
	@ex: Google Chrome など。
	
＠導入
	適当なところに WinChild を解凍してください。
	
＠除去
	WinChild のフォルダごと削除してください。
	
	レジストリなどは弄っていません。
	
＠操作方法
	WinChild が起動されている状態で、他のウィンドウのタイトルバー上で
	[Ctrl] + [Shift] + [Enter] を押すと、そのウィンドウを「捕獲」できます。
	一度「捕獲」したウィンドウは、元に戻せません (元に戻した後に正常に動作する
	かどうかわからないため)。
	
	WinChild を最小化すると、タスクトレイに収まります。ウィンドウのような形の
	アイコンを、左右どちらかでクリックすると、元の大きさに戻ります。
	
＠実装詳細
	対象ウィンドウを、タブページに割り当たウィンドウの子にしています。
	( Win32API: WindowFromPoint(), SetParent() )
	
＠ライセンス
