# HSP3 Plugins
HSP3拡張プラグインのソースコードをまとめたレポジトリー

## プラグイン
### [hsp3opex](hsp3opex)
短絡的な演算を提供するプラグイン。

### [var_modcmd](var_modcmd)
ユーザ定義コマンドを値として持つ型を提供するプラグイン。

### [valptr](valptr)
「任意の値」型を提供するプラグイン。
