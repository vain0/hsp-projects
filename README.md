# valptr## 概要値へのポインタを生成するHSP3拡張プラグインです。関数へ任意の型の値を渡すときなどに使えます。ポインタの生存期間は長くないので、早めに使ってください。詳しくは [サンプル](package) を参照。## 使い方* HPIファイル [valptr.hpi](package/valptr.hpi) を、スクリプトと同じフォルダか、HSPのフォルダにコピーします。  * 実行形式を配布する場合は、それと同じ所にこのHPIもコピーする必要があります。* スクリプトに ``#include "valptr.as"`` と書くと、各種コマンドが使用できます。## 参照* プログラ広場 <http://prograpark.ninja-web.net>:  * サポートページです。意見や要望、バグ報告などはここの掲示板までお願いします。  * HSPプラグイン講座もあります。* HSPTV! <http://hsp.tv>  * HSP3の公式サイトです。## 更新履歴#### 2011/Apr/10* ver 1.1  * new_valptr() で可変長の値を使用できるようにした (型は統一)。  * fromValptr → value_get に改名。  * value_get() の第2引数で、要素番号を取れるようにした。#### 2011 02/06 (Sun)* ver 1.0 完成、公開。
