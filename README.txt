ZeoFive は、手軽に遊べる新感覚カードゲームです。

現在、CPUとの対戦機能のみ実装しています。

デッキは、Zeo_Decks フォルダにサンプルデッキが6つある他、
DeckEditor で自由に作成することが可能です。
