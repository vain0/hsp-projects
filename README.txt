＠概要
	X Eaters は、唯食者、つまりただ一つのモノしか食べない者たちに、適切に餌を
	やるゲームです。ゲーム開始時の X Eaters はまだ子供なので( 外見に変化はない )、
	あまりたくさん食事をする必要がありません。しかし、中盤以降になると、成長して
	食欲旺盛になります。止めどなくえさを与えなければ、彼らは生きていくことが出来
	ません。
	
	また、唯食者は気まぐれ屋さんです。彼らの好みは、すぐに変わってしまいます。
	それに、同じ場所でじっとしていることも出来ないのです。
	
	さぁ、そんな彼らをひたすら育ててみましょう！
	
＠操作方法
	・タイトル画面
		ゲームを起動したときは、タイトル画面になります。
		「Press [Enter] to Begin!」
		つまり、始めるために [Enter] を押してください。
		
	・ゲーム画面
		左にある４つの顔が、唯食者たちです。
		彼らの口につながっているパイプは、
		彼らにえさを渡すための道具「餌管」です。
		
		食事中は口をもぐもぐ ^～^ させています。
		
		餌管のなかにある四角いものが、唯食者たちの餌になります。唯食者たちは、
		自分の好みの色のものしか食べることができません。カーソルキーの↑↓を
		使って、与える唯食者を変え、うまくすべての餌を唯食者に正しく与えて
		ください。
		途中で、通常より丸みを帯びた「ゴミ餌」というものが登場することがあります。
		このゴミ餌は唯食者にとって極めて有害なため、唯食者に与えてはいけません。
		これを餌管から取り除くには、餌管の一番左がこれになったときに [Space] を
		押してやってください。
		
		画面下の方にあるバーは、スコア・バーです。
		ここに得点が表示されます。
		
	・ゲームオーバー画面
		残念ながら、唯食者に食べられないえさを与えてしまった場合、
		ゲームオーバーとなり、ゲームオーバー画面になります。
		
		GameOver... の次の行の数値は、最終的なスコア、得点です。
		次の行にあるとおり、
		「Press [Enter] to continue.」
		つまり「再挑戦するために [Enter] キーを押してください」により、
		[Enter] キーを押すことでタイトル画面に戻ることが出来ます。
		
	・終了
		いつでも[×]ボタンで閉じることができます。
		ただし、終了の確認は行われません。

＠更新履歴 (開発日記)
2008 10/31 (Fri)
	・このゲームを思いつく。
	・とりあえず完成。恐ろしいほど単調。つまらん。
	・タイトルを「カラー・マッチ」から、「X Eaters」へ変更。
	　左のやつを唯食者に変更。(あんま変わっとらんが……)
	・色替え、シャッフル、高速化などの機能を追加。
	　→ バージョン 1.00
	　難易度急上昇。
	・HSPコンテスト2008 (ショート部門) に応募。
	　締め切り８分前。
	  
2008 11/01 (Sat)
	・ショート部門用に、640×480の版を作成。
	　ランキング機能追加。
	・唯食者の好む色を、カラーテーブルから選ぶように変更。
	　以前では、輝度 1 の差など、人間では識別不可能な違いで
	　ミスにされていた。
	・唯食者の好む色と、えさの色のマッチングを、実際のCOLORREF値で
	　比較するよう変更。
	　以前では、対象の唯食者IDとえさの唯食者IDで比較していたので、
	　同じ色を好む唯食者でもミスにされていた。( えさと唯食者を一対一に割り当てていた )
	・色替え、シャッフルなどの発生率の調整。
	　連続で起こること確率を低くした。( 可能性はある )
	・難易度( 速度 )を４段階に分割。
	・ダミー発生率を低下させた。
	・バージョン 1.01。
	
2008 11/03 (Mon)
	・スコアバーにレベルを表示する機能を追加。
	　本物のステータスバーにしても問題ないかも。
	・バージョン 1.02。
	
2009 08/03 (Mon)
	・たびたびゴミ餌が流れてくるようにした。
	・HSPコンテスト2009に改めて提出 (バグったままじゃ嫌だからね)。
	
2009 08/04 (Tue)
	・XEater の口と餌の当たり判定が意味不明だったので、
		餌の位置を配列で持つように修整。( 今までは意味不明な式で算出していた )
	・ゴミ餌じゃない餌を弾き飛ばそうとしてもゲームオーバーにならないように変更。
		・難易度を下げる目的
		・食後イベントが発生すると、次の2つの餌が消滅する仕様になってるため、
			「餌、ゴミ餌」の順で、短い隙間しかないときに、
			餌を食べた瞬間 [Space] 打鍵
			→反応する前にイベント＋ゴミ消滅
			→[Space]感知
			→ゲームオーバー
			という不可抗力な事態に陥るため。
			餌を弾こうとしたペナルティは、スコア4点減点。
